그리고 겟허브, 태크맥 과같은 처음보는 프로그램들을 이용해 보았다.

+ 데이터를 분류한다.
+  클래스를 만든다. <--라밸링
+  분류한 것들을 넣어 학숩시킨다.
+ 태스트해본다.
+ 내보낸다.
