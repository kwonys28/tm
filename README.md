## **느낀점**

여기서 나는 인공지능의작동 방식을 배우고 이를 간단하게 채험해보는 시간을 가졌다. 

바로 인공지능 처럼 사진을 주어 학습시키고 이를 다른 임의의 사진을 가져다 주어 누구인지 판별 할 수 있게 해주는 프로그램이다.

그리고 겟허브, 태크맥 과같은 처음보는 프로그램들을 이용해 보았다.  
+ 힉습데이터를 모은다.
+ 데이터를 분류한다.
+  클래스를 만든다. <--라밸링
+  분류한 것들을 넣어 학습시킨다.
+ 태스트해본다.
+ 내보낸다.
오늘 이런 것 들을 해보면서 생각보다 코딩이 힘들었던 것 같았다.
